### Selected work

**ARLAMX** &nbsp;·&nbsp; *private* &nbsp;·&nbsp; *in development*
Deep RL attitude controller (PPO / SAC) for a 3U CubeSat with domain randomization. Comparable to classical SMC under known dynamics; more robust under model uncertainty.

**TAVIS**
Ball-tracking system for Stokes-flow viscometry. YOLO detection, PyQt5 GUI, Kalman-filtered velocity estimation, Faxén wall corrections.
